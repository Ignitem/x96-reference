# Embedded Comments

## Encoding

| Opcode              | Instruction   | Description                     |
|---------------------|---------------|---------------------------------|
| `0F 0F ascii 0F 0F` | `<> ascii <>` | Embedded comments in the binary |

## Usages

Embedded comments can be used for:

- Copyright notices
- License informations
- Debugging informations
- Warnings
- Metadata
